# Bank Management System (Python Project)

## 📌 Project Overview

This is a simple **Bank Management System** developed using Python.\
It is a menu-driven program that allows users to perform basic banking
operations like creating an account, checking balance, depositing and
withdrawing money.

------------------------------------------------------------------------

## 🎯 Objectives

-   To understand basic Python programming concepts\
-   To implement real-world banking operations\
-   To improve logic building and problem-solving skills

------------------------------------------------------------------------

## ⚙️ Features

-   Create a new account\
-   Check account balance\
-   Credit (Deposit) money\
-   Debit (Withdraw) money\
-   Close account\
-   Menu-driven interface

------------------------------------------------------------------------

## 🛠️ Technologies Used

-   Python\
-   Command Line Interface (CLI)\
-   VS Code (Editor)

------------------------------------------------------------------------

## ▶️ How to Run the Project

1.  Install Python on your system\
2.  Open the project file in VS Code\
3.  Run the Python file\
4.  Follow on-screen menu instructions

------------------------------------------------------------------------

## 🔄 Working

-   User selects an option from menu\
-   Program calls the respective function\
-   Data is stored and updated in dictionary\
-   Loop continues until exit

------------------------------------------------------------------------

## 📉 Limitations

-   No database (data not saved permanently)\
-   No security/authentication\
-   No GUI (only CLI based)
